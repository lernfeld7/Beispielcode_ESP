# Ventilator

Dieses Beispiel zeigt wie man den Ventilator mittels [Pulsweitenmodulation](https://www.elektronik-kompendium.de/sites/kom/0401111.htm) ansteuert. Es benötigt keine zusätzlichen Bibliotheken.

Der Signalunterschied zwischen den beiden Eingangskanälen bestimmt Richtung und Geschwindigkeit des Ventilatormotors.
