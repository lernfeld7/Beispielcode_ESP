# Beispiel-Code für den ESP32

Diese Code-Beispiele können in eigene Projekte eingebaut werden. **Lies für ein bessers Verständnis die Beschreibungen zu den Beispielen und die im Quellcode enthaltenen Kommentare.**

- MQTT-Kommunikation mit PubSubClient
- Digitaler Input und Output: Buttons, LED, Relais
- Display LCD 1206
- PIR-Bewegungsmelder
- Buzzer zur Ausgabe von Signaltönen mit Morse-Code-Beispiel
- Servo zur Bewegung von Fenster und Tür
- Ventilator mit variabler Geschwindigkeit und wechselnder Richtung
