# Display LCD 1602

Dieses Beispiel zeigt die Verwendung eines LCD 1602-Displays an der I²C-Schnittstelle. Es wird die Bibliothek LiquidCrystal I2C von Frank Brabander benötigt, welche über die Bibliotheksverwaltung installiert werden kann. Das Display wird an die I²C-Schnittstelle (SCL/SDA) angeschlossen.
